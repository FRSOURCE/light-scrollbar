Light-scrollbar

Small light custom scrollbar

Light-scrollbar plugin is based on one wrap element and psuedo elements that creates scrollbar for X and Y axis, this provides the smallest alteration of DOM possible.

Setup

Easy setup, just one line to initilize whole plugin, bother no longer to understand how to replace your scrollbar.

API

Straightforward API, easy to understand where most of customization.

Light

Super light plugin that weights around 2500B, with this size works exactly like native scrollbar but with possibility of customization.
